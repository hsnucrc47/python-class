字串轉換方法

傳回轉換成大寫的字串

a.upper()：傳回 a的大寫

a = "Hello, World."
b = a.upper () # b 的值為 "HELLO, WORLD."

傳回轉換成小寫的字串

a.lower()：傳回 a的小寫

a = "Hello, World."
b = a.lower () # b 的值為 "hello, world."

大小寫互換

a.swapcase()

a = "Hello, World."
b = a.lower () # b 的值為 "hELLO, wORLD."

取代

a.replace([舊的參數], [新的參數])

a = "Hello, World."
b = a.replace ("World", "Python") # b 的值為 "Hello, Python."

第一個字元轉成大寫

a.capitalize()

a = "never mind."
b = a.capitalize() # b 的值為 "Never mind."

每個單字第一個字元轉成大寫

a.title()

a = "never mind."
b = a.title() # b 的值為 "Never Mind."
