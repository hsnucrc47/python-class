第一題　馬蹄鐵（horseshoe）

出題者：coix


參考答案

【解法一】 PYTHON 
k, n = map(int, input().split()) 
for _ in range(k):
    l = list(map(int, input().split()))
    max_value = max(l)
    print(max_value)
【解法二】 PYTHON 
k, n = map(int, input().split()) 
for _ in range(k):
    l = list(map(int, input().split()))
    max_value = 0 # 每次測試前重製
    for i in range(n):
        if max_value < l[i]: # 如果第 i 個數字大於目前的最大值
            max_value = l[i] # 將最大值設為第 i 個數字
    print(max_value) # 將每個最大值放進 ans 串列裡

解說


根據解法一，要先利用迴圈執行 k 次找最大值。 l 為我們輸入進來的陣列。我們可以藉由 max(l) 找出最大值，再將其輸出。

值得注意的是，解法一不會用到輸入的 n ，如果要用的話，可以參考解法二，設定一個變數（max_value）預設為零，再逐一比較，如果比 max_value 還要大，就將 max_value 設為此數字。


講義連結

第二題 底線蛇（_snake）

出題者：Jasmine


參考答案

【解法一】 PYTHON 
T = int(input())
for _ in range(T):
    string = input()
    parts = string.split('_') # 分割
    reversed_parts = [] # 創空串列用來放反轉後的結果
    for part in parts:
        reversed_part = part[::-1] # 反轉
        reversed_parts.append(reversed_part) # 把反轉後的字串放回去
    for i in range(len(reversed_parts)): # 把反轉後的字串逐一輸出
        print(reversed_parts[i], end="")
        if i < len(reversed_parts) - 1: # 在字串後加上 _ 除非是最後一個字串
            print("_", end="")
    print()
【解法二】 PYTHON 
T = int(input())
for _ in range(T):
    string = input()
    reversed_parts = [s[::-1] for s in string.split('_')]
    result_string = '_'.join(reversed_parts)
    print(result_string)

解說

在解法一中，這題要設兩層三個迴圈，第一層迴圈設為執行 T 次，再經由第二個迴圈處理反轉，最後用第三個迴圈輸出答案。

在第一層迴圈中，我們利用 .split('_') 將輸入進來的字串分割成串列，並創一個空串列（ reversed_parts ）用來放反轉後的結果。

第二層迴圈負責反轉字串，利用 for 迴圈遍歷每個字串，再利用 [::-1] 反轉字串，最後將其儲存至串列 reversed_parts 中。

最後第三迴圈會將 reversed_parts 裡反轉後的字串逐一印出，並在間隔放上 _ 。當本組測試結束時，使用 print() 換行以區分下一組。

解法二是比較簡潔的做法，不過會用到一些沒教過的東西。 [s[::-1] for s in string.split('_')] 是將字串直接分割成串列並反轉； '_'.join(reversed_parts) 是將串列利用 _ 連接起來變回串列。

第三題　分割（divide）

出題者：昱山


參考答案

PYTHON 
def count_partitions(n, max_part):
    if n == 0:
        return 1
    if n < 0 or max_part == 0:
        return 0
    return (count_partitions(n, max_part - 1) + count_partitions(n - max_part, max_part))
n = int(input())
print(count_partitions(n, n))

解說

這次最難的一題由我出，非常考驗自己的邏輯是否清楚。 但不知道為什麼第三題答對比例比第二題高一點，甚至有人不知道為什麼可以只通過部分的測資。明明第二題比較容易想啊。

我們可以利用遞迴解，選定長度後，我們要逐一比較當最長積木多長時要用什麼積木，因此我們在函式放兩個參數。 n 代表還剩下多長，如果最後 n = 0 表示這組組合可以，但若最後 n < 0 表示這組組合不行。 max_part 代表我們在組合當中所使用的最大積木，如果 max_part = 0 也是不行的。

遞迴的邏輯是先從最大的積木（也就是 max_part = n = 最大值 時）算起，如果成功讓 n = 0 就回傳 1 ， 但如果出現 n < 0 或 max_part = 0 就回傳 0 。上述都沒發生的話就繼續分割，有兩個路線，一種是將最大的積木長度減一分析。另一種是保留最大的積木長度，剩下長度為 n 減掉最大的積木長度，分析第二長的積木與其組合，以此類推。
