第一題　光（Light）

參考答案


#輸入
a1, n1, a2, n2 = input().split()
a1, n1 = float(a1), int(n1)
a2, n2 = float(a2), int(n2)

#計算數值
num1 = a1 * (10 ** n1)
num2 = a2 * (10 ** n2)

#比大小
if num1 > num2:
    print(a1,"* 10 ^", n1)
elif num2 > num1:
    print(a2, "* 10 ^", n2)
else:
    print("Same")

解說

第一題要利用條件判斷比大小，由於我們沒有練習過一行輸入多個變數，因此直接給大家第一行到第三行的輸入方式。

處理好輸入後，我們就得思考如何運用這些變數。首先，我們分別計算兩數的值，令第一個數為 num1 ；第二個數為 num2 。 再來用到 if 條件判斷，如果 num1 大於 num2 ，印出「a1 * 10 ^ n1」，否則如果 num1 小於 num2 ，印出「a2 * 10 ^ n2」。 如果前面兩個條件都不符合，就印出「Same」。要注意印出科學記號時符號與數字之間都要有空格。



第二題 血壓（blood pressure）

參考答案


s = int(input()) # 收縮壓
d = int(input()) # 舒張壓
if s >= 160 or d >= 100:
    print("第二期高血壓")
elif 140 <= s <= 159 or 90 <= d <= 99:
    print("第一期高血壓")
elif 120 <= s <= 139 or 80 <= d <= 89:
    print("高血壓前期")
else:
    print("正常血壓")

解說

這題是我出的，也是考大家條件判斷，只要寫出正確的判斷就能答對，不過這題有個小陷阱。 如果條件判斷順序顛倒，從正常血壓的條件開始判斷、第二期高血壓放在else就會出錯。 因為條件裡面有「or」，假設收縮壓為 170 、 舒張壓為 95 ，應該判斷為「第二期高血壓」。 但若條件判斷順序顛倒，會被低估為「第一期高血壓」，要特別注意。



第三題 種樹（plant trees）

參考答案


n = int(input())
k = int(input())

# 樹的葉子
for i in range(2, n + 2):
    for j in range(1, i + 1):
        print('B' * (n + 1 - j) + 'A' * (j * 2 - 1) + 'B' * (n + 1 - j))

# 樹幹
for _ in range(k):
    print('B' * n + 'C' + 'B' * n)

解說

這題是魔王題，兩班沒有任何一個人答對，真的很有難度，答不出來也不用太灰心。

這題要用迴圈解，因為有 n 層且最高層為兩行，所以範圍為 range(2, n + 2) ，要記得 i 不會碰到 n+2 所以 i 最大為 n+1 。

再來我們在迴圈內層設另一個迴圈處理 A 的遞增與 B 的遞減，範圍為 range(1, i + 1)，因為每一層遞增與遞減次數為該層層數 （ i ），所以讓變數 j 從 1 跑到 i 。

最困難的部分是寫出 A 與 B 的輸出。兩側 B 的數量會從 n 開始遞減且公差為 -1 ； j 從 1 開始越變越大，因此 B 的數量公式為 (n + 1 - j) ，且左右兩邊都要有 B 。中間的 A 會從 n 開始遞增且公差為 2 ； j 從 1 開始越變越大，因此 A 的數量公式為 (j * 2 - 1)。

最後的樹幹就比較輕鬆了，左右兩邊 B 的數量分別都是 n 。中間則是夾著一個 C 。

我覺得這題非常值得大家深入研究，因為 APCS 很常考這種將外層 for 迴圈的變數 i 放到內層 for 迴圈 的 range() 。看完解說後，如果你可以全部理解並自己重新做一遍，那我覺得你可以輕鬆應對 APCS 的迴圈題目了。
