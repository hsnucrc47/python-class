time 模組

匯入模組

import time

暫停程式執行

time.sleep(seconds)：讓程式暫停指定秒數。

print("開始")
time.sleep(2)
print("結束")  # 會延遲 2 秒輸出

取得現在時間（時間戳）

time.time()：回傳從 1970 年 1 月 1 日到現在的秒數（浮點數）。

print(time.time())  # 例如 1762580911.319975

取得結構化時間

time.localtime()：取得目前本地時間，回傳 time.struct_time。

t = time.localtime()
print(t.tm_year, t.tm_mon, t.tm_mday)

time.gmtime()：取得 UTC（格林威治）時間。

格式化時間

time.strftime(format, t)：將時間物件轉為字串。

now = time.localtime()
print(time.strftime("%Y-%m-%d %H:%M:%S", now))

time.strptime(string, format)：將字串轉回時間物件。

t = time.strptime("2025-11-08 13:30:00", "%Y-%m-%d %H:%M:%S")
print(t)

時間與秒數轉換

time.mktime(t)：將 struct_time 轉為秒數（timestamp）。

ts = time.mktime(time.localtime())
print(ts)

time.ctime(secs)：將秒數轉為易讀格式字串。

print(time.ctime(1731052800))  # Sat Nov  8 13:00:00 2025

計算程式執行時間

可用 time.time() 或 time.perf_counter() 量測程式執行所需時間。

start = time.perf_counter()
for i in range(1000000):
    pass
end = time.perf_counter()
print("執行時間:", end - start)

常見應用範例

顯示目前時間

print(time.strftime("%Y/%m/%d %H:%M:%S", time.localtime()))

每秒倒數計時

for i in range(5, 0, -1):
    print(i)
    time.sleep(1)
print("時間到")

量測程式效能

start = time.time()
sum(range(1000000))
print("耗時:", time.time() - start, "秒")
