FFmpeg


FFmpeg 是一個能夠解碼、編碼、轉碼、封裝、解封裝等影音處理的多媒體格式。


前置作業


1️⃣下載執行檔

進入網站：

FFmpeg Build - binaries for Windows

下載 release builds 中的 ffmpeg-release-essentials.zip 並解壓縮。



2️⃣新增路徑

在 C:\ 建立資料夾 ffmpeg ，並將 ffmpeg-release-essentials 資料夾中的 bin 資料夾移動到 ffmpeg 資料夾中，確認符合以下路徑：

OS 
C:\ffmpeg\bin\FFmpeg.exe
3️⃣設定環境變數

按 Win 鍵搜尋「編輯系統環境變數」，點選「編輯系統環境變數」。彈出視窗後按「環境變數」。

在「環境變數」視窗中點擊系統變數中的 Path ，然後按「編輯」。



在「編輯系統環境變數」中按新增，貼上剛才的路徑，然後就連按確定把兩個視窗都關掉。




這樣就完成了，可以開啟終端機輸入以下指令檢查（如果已經開的話要重開）。如果出現了版本資訊那恭喜你成功設定好環境變數了。

BASH 
ffmpeg -version
4️⃣安裝套件

在終端機輸入以下指令：

BASH 
pip install ffmpeg-python

程式實作


請先確認影片名稱與根目錄路徑：



影片轉檔
PYTHON 
import ffmpeg

(
    ffmpeg
    .input('input.mp4')
    .output('output.avi')
    .run()
)
擷取音訊
PYTHON 
import ffmpeg
(
    ffmpeg.
    input('video.mp4')
    .output('audio.mp3')
    .run()
)
擷取圖片
PYTHON 
import ffmpeg
(
    ffmpeg
    .input('video.mp4', ss=5) # 第 5 秒截圖
    .output('frame.jpg', vframes=1)
    .run()
)
影片裁切
PYTHON 
import ffmpeg
(
    ffmpeg
    .input('input.mp4', ss=10, t=5) # 從第 10 秒開始剪 5 秒
    .output('clip.mp4')
    .run()
)
改解析度
PYTHON 
import ffmpeg
(
    ffmpeg
    .input('input.mp4')
    .output('output.mp4', s='1280x720') # 將解析度改為 1280x720
    .run()
)
加上字幕
PYTHON 
import ffmpeg
(
    ffmpeg
    .input('input.mp4')
    .output(
        'output.mp4',
        vf="subtitles=sub.srt" # 字幕檔
    )
    .run()
)
串流 / 即時處理

將錄好的影片即時直播在直播平臺（以 YouTube 為例）。

PYTHON 
import ffmpeg
(
    ffmpeg
    .input('input.mp4', re=None)  # 模擬即時播放
    .output(
        'rtmp://a.rtmp.youtube.com/live2/你的串流金鑰',
        format='flv',           # 串流格式
        vcodec='libx264',       # 視訊編碼器
        acodec='aac',           # 音訊編碼器
        video_bitrate='3000k',  # 視訊位元率
        audio_bitrate='128k',   # 音訊位元率
        r=30,                   # FPS 
        s='1280x720'            # 解析度
    )
    .run()
)
