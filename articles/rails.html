火車調度問題



📘 題目連結

參考答案：



while True:
    n = int(input()) #有 n 節車廂

    #沒有火車的話直接結束
    if n == 0: 
        break
    
    while True:
        
        #想要的排列方式
        a = list(map(int, input().split()))

        #當遇到僅含有一個 0 的一列，代表該組測試資料結束。
        if a[0] == 0:
            break

        cur = 0 #目前要出站的車廂
        stack = [] #以 stack 模擬車站

        #模擬火車進站
        for i in range(1, n + 1):
            stack.append(i) #車廂進站

            #如果最上面的車廂剛好是我們要的順序 a[cur] ，那就讓它出站（pop）
            while stack and stack[-1] == a[cur]:
                stack.pop()
                cur += 1

        #判斷火車能否以一特定的排列方式在B方向的鐵軌上
        if not stack:
            print("Yes")
        else:
            print("No")
    print()


解說：



第一層 while True:

while True:
    n = int(input())
    if n == 0:
        break

控制「有幾個車廂」的迴圈，沒有的話直接結束。





第二層 while True:

    while True:
        a = list(map(int, input().split()))
        if a[0] == 0:
            break

處理每一組排列的測試。若為 0 表示本次測試結束。



主體模擬

        cur = 0
        stack = []
        for i in range(1, n + 1):
            stack.append(i)
            while stack and stack[-1] == a[cur]:
                stack.pop()
                cur += 1

模擬車廂依序進入。 stack 為車站 ； cur 為目前要出站的車廂，預設為 0 （第一個要出去的），負責當 a 的索引值。當發現進入 stack 的車廂剛好是現在要出去的車廂時，讓車廂出去。並改偵測下一個車廂（cur += 1）。



判斷結果

        if not stack:
            print("Yes")
        else:
            print("No")

如果跑完後沒有車廂留在車站，表示能夠成功；如果跑完後還有車廂留在車站，表示不可能成功。
