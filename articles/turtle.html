turtle 模組


turtle 是 Python 內建的一個繪圖模組，非常適合初學者學習程式邏輯與幾何圖形。


無需安裝


這是 Python 標準函式庫的一部分，所以不需要執行 pip install，直接 import 即可使用！


程式碼範例：畫一個正方形


PYTHON 
import turtle # 載入海龜繪圖模組

t = turtle.Turtle() # 建立一隻海龜
t.shape("turtle")   # 把圖示改成海龜形狀
t.color("orange")   # 設定顏色

for i in range(4):  # 重複四次
    t.forward(100)  # 向前走 100 像素
    t.left(90)      # 向左轉 90 度

turtle.done()       # 繪圖結束，視窗不會立即關閉

透過簡單的 forward (前進) 與 left (左轉) 指令，我們就能在畫面上指揮海龜畫出各種繽紛的圖案。
