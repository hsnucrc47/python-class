random 模組

匯入模組

import random

生成隨機整數

random.randint(a, b)：產生介於 a 和 b（含）的整數。

random.randint(1, 6)  # 模擬骰子點數

生成隨機浮點數

random.random()：產生 0.0 ≤ x < 1.0 的浮點數。

random.random()  # 例如 0.374

random.uniform(a, b)：產生介於 a 和 b 的浮點數。

random.uniform(5, 10)  # 例如 7.328

從序列中隨機選取元素

random.choice(seq)：從序列（串列、字串等）中隨機選取一個元素。

random.choice(["剪刀", "石頭", "布"])
random.choice("ABCDE")  # 例如 'C'

隨機抽樣

random.sample(seq, k)：從序列中選取 k 個不重複元素（回傳新串列）。

random.sample(range(10), 3)  # 例如 [1, 7, 4]

random.choices(seq, k=n)：從序列中選取 k 個可重複元素。

random.choices("ABC", k=5)  # 例如 ['B', 'A', 'C', 'C', 'B']

打亂順序

random.shuffle(list)：將串列內的元素隨機重新排序（直接改變原串列）。

cards = ["A", "B", "C", "D"]
random.shuffle(cards)
print(cards)  # 例如 ['C', 'A', 'D', 'B']

設定隨機種子（重現結果）

random.seed(x)：設定隨機數種子，使結果可重現。

random.seed(10)
print(random.random())  # 每次執行都會產生相同結果

常見應用範例

模擬擲硬幣

coin = random.choice(["正面", "反面"])

產生隨機密碼

import string
pw = ''.join(random.choices(string.ascii_letters + string.digits, k=8))
print(pw)

模擬抽獎

winners = random.sample(["小明", "小華", "小美", "小王"], 2)
print(winners)
